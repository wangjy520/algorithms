## D
- 前缀和 or 异或前缀和
- 对于奇偶性变换这种问题，特别适用于异或操作，0表示偶数，1表示计数。判断多个数的和的奇偶性只需要将这些数进行异或操作（最后还要异或上0）即可
- 如果是多个数的乘积，则使用与操作（最后与1）
